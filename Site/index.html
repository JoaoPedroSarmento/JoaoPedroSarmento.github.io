back-end x Front-end


Front-end serve para criar a interface de um site , geralmente usando html,css,JavaScript, Sass e algumas frameworks como : Bulma e bootstrap .


Back-end serve para criar a parte não visual, parte que interage com os outros servidores e que processa regras de negócio .

Frameworks : São facilitadores no desenvolvimento de diversas aplicações, pois traz consigo classes e funções predefinidas , facilitando a criação de páginas elegantes e responsivas .

O que é um servidor web ?

Servido web é um software que fica instalado em um computador na rede e serve para receber as requisições de um browser e responder com arquivos correspondente do site solicitado .

Protocolo http : hypertext transfer protocol , isto é um ambiente utilizado para que um cliente web e o servidor web se entendam .
