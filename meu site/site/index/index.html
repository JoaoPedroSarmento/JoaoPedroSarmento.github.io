    HTML


    HTML ( HyperText Markup Language) é uma linguagem de marcação , mas o que é uma linguagem de marcação ? Em um arquivo de texto, como um arquivo HTML, os elementos são marcados utilizando tags, que explicam o objetivo dessa parte do conteúdo.




    CSS


    CSS ( Cascading Style Sheets ) o css adiciona um estilo a um documento web,pode ser incluido diretamente nas tags do html ou pode ser incluido atravéz do <style> ou do <link rel="stylesheet" href="link.css">.
