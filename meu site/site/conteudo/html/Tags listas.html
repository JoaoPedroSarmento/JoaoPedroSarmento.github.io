    Listas ordenadas ou numeradas :


    As listas ordenadas ou numeradas são reprersentadas pela tag <ol>, essa e a tag de abrimento de uma lista numerada.
    Para escrever dentro de uma lista necessitamos da tag <li> para indicar que é uma lista,abaixo uma lista numerada :


    1. Primeiro
    2. Segundo
    3. terceiro
    4. Etc

    As listas não ordenadas são reprersentadas pela tag <ul>, essa e a tag de abrimento de uma lista não ordenada .
    abaixo uma lista não ordenada :

    1. Primeiro
    2. Segundo
    3. terceiro
    4. Etc
