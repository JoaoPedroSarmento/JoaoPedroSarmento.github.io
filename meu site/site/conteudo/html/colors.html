    • Seletores de cores no css


    A ferramenta de cores facilita a personalizar um site. Serão mostrado diferentes cores e formas de escolher uma cor.


    • Escolher a cor em inglês (red,yellow,black,etc...).
    • Código hexadecimal (#e43f5a,#ffffff, #ababab,etc...).
    • Cores rgb (red, green e blue) < significado, as cores são atingidas com a mistura das três cores primárias em um número definido numa escala de 0 a 255.
    • Rgba :"RGBA é o sistema de cores formado pelas cores vermelho, verde, azul e pelo canal alfa. O sistema permite exibir todas as cores do sistema RGB e a utilização da transparência de imagem".
    • Hsl : "Os parâmetros de cor utilizados nesse sistema são a tonalidade, a saturação e o brilho."
    • Hsla : "Os valores de cor HSLA são uma extensão dos valores de cor HSL com um canal alfa - que especifica a opacidade da cor."

    A diferença entre esses seletores :


    10 css : background-color: black; (cor de fundo preta) idicando qual será a cor de fundo
    apenas com a palavra chave em inglês

    2 0 css código hexadecimal : background-color: #000;(cor de fundo preta em hexadecimal).

    3 0 css rgb : background-color: rgb(0, 0, 0); (cor de fundo preta em rgb).

    4 0 css rgba : background-color: rgba(0, 0, 0, 1);(cor de fundo preta em rgba).

    5 0 css hsl : background-color: hsl( 0.00 0.00 0.00); ( cor de fundo preta em hsl )

    6 0 css hsla : background-color:hsla(0.00 0.00 0.00) ( cor de fundo preta em hsla )
